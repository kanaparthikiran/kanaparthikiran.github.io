## Kiran Kanaparthi

**Principal Engineer — Enterprise Agentic AI Platforms, Distributed Systems &
Developer Infrastructure.** Cupertino, CA.

I build production agentic AI systems that take real, audited write-actions
inside enterprise systems — multi-agent orchestration, MCP tool frameworks,
LLM gateway integration, and the evaluation/telemetry discipline that keeps
them honest. Twenty-plus years before that on large-scale distributed systems
and developer infrastructure.

This site is where I think out loud about agent design, tool surfaces, and
the engineering practice around AI in production.

### Public projects

- **[multi-agent-langgraph-demo](https://github.com/kanaparthikiran/multi-agent-langgraph-demo)** —
  reference implementation of the planner / executor / reviewer pattern on
  LangGraph + NVIDIA NIM, with bounded iteration and audit-trail state.
- **[rag-notes](https://github.com/kanaparthikiran/rag-notes)** —
  end-to-end RAG pipeline (chunk → embed → FAISS → grounded LLM) with an
  LLM-as-judge eval harness on NVIDIA NIM. Verified 1.0 recall@k and 1.0
  answer-match on the test corpus.
- **[mcp-tasks-server](https://github.com/kanaparthikiran/mcp-tasks-server)** —
  Model Context Protocol server with five typed tools over SQLite;
  demonstrates audited write-action surface for agent tool frameworks.
